## etc
일반적으로 리액트에서는 비동기 통신을 이용해서 서버에 접근하여 데이터를 가져오도록 코딩
,props는 변경할 수 없는 데이터를 명시할 때 사용

npm install -g react-scripts

## mysql library installation
npm install --save mysql
